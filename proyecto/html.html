HTML: La Estructura Fundamental de la Web

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Es el esqueleto de tu sitio, definiendo la estructura y el contenido. Cada elemento HTML, o "etiqueta", tiene un propósito específico y organiza el texto, las imágenes, los enlaces y otros elementos para que el navegador los interprete correctamente.

Estructura Básica de un Documento HTML

Todo documento HTML comienza con una declaración de tipo de documento y se divide en dos secciones principales: el `<head>` y el `<body>`.

  • <!DOCTYPE html>: Declara el tipo de documento y la versión de HTML (HTML5).
  • <html lang="es">: El elemento raíz de la página, indicando el idioma principal.
  • <head>: Contiene metadatos sobre la página (no visibles en el navegador), como el título, la codificación de caracteres, enlaces a hojas de estilo, etc.
  • <meta charset="UTF-8">: Define la codificación de caracteres para que el texto se muestre correctamente.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Esencial para el diseño responsivo, controla el ancho del viewport y el zoom inicial.
  • <title>: El texto que aparece en la pestaña del navegador.
  • <link rel="stylesheet" href="styles.css">: Enlaza tu hoja de estilos CSS externa.
  • <body>: Contiene todo el contenido visible de la página web.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de mi Página</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <!-- Aquí va todo el contenido visible -->
</body>
</html>

Etiquetas HTML Semánticas (HTML5)

HTML5 introdujo etiquetas con significado semántico que describen la función del contenido, mejorando la accesibilidad y el SEO. Evita usar `<div>` para todo cuando haya una etiqueta más específica.

  • <header>: Contenido introductorio o de navegación (ej. logo, título del sitio, barra de navegación).
  • <nav>: Un grupo de enlaces de navegación (menú principal).
  • <main>: El contenido principal y dominante de la página, único por documento.
  • <article>: Contenido independiente y autocontenido (ej. entrada de blog, artículo de noticias).
  • <section>: Un agrupamiento temático de contenido, a menudo con un título (ej. capítulos, introducciones).
  • <aside>: Contenido indirectamente relacionado con el contenido principal, como barras laterales o bloques de publicidad.
  • <footer>: Información sobre la sección contenedora o el documento (ej. derechos de autor, información de contacto).

Elementos de Texto y Contenido

Estas son algunas de las etiquetas más comunes para estructurar tu texto:

  • Encabezados: <h1> a <h6> (del más importante al menos importante). Solo debe haber un `<h1>` por página.
  • Párrafos: <p> para bloques de texto.
  • Listas:
    • <ul>: Lista desordenada (ítems con viñetas). Usa <li> para cada ítem.
    • <ol>: Lista ordenada (ítems numerados). Usa <li> para cada ítem.
    • <dl>: Lista de definición (para términos y sus descripciones). Usa <dt> para el término y <dd> para la descripción.
  • Énfasis y Resaltado:
    • <strong>: Contenido de gran importancia (visualmente negrita por defecto).
    • <em>: Énfasis (visualmente cursiva por defecto).
    • <b>: Texto en negrita sin especial énfasis.
    • <i>: Texto en cursiva para un estado de ánimo o voz alternativa.
  • Enlaces: <a href="url">Texto del enlace</a> para crear hipervínculos. El atributo `href` es crucial. Puedes usar `target="_blank"` para abrir en una nueva pestaña.
  • Saltos de línea: <br> para un salto de línea (no de bloque).
  • Líneas horizontales: <hr> para un cambio temático en el contenido.

<h2>Mi Título Principal</h2>
<p>Este es un <strong>párrafo</strong> con texto importante.</p>

<ul>
    <li>Primer ítem</li>
    <li>Segundo ítem</li>
</ul>

<p>Visita <a href="https://www.google.com" target="_blank">Google</a> para más información.</p>

Trabajando con Imágenes

Las imágenes son un componente visual clave de cualquier página web. La etiqueta `<img>` es fundamental.

  • <img src="ruta/a/imagen.jpg" alt="Descripción de la imagen">: La etiqueta para insertar imágenes.
  • `src` (source): La ruta al archivo de la imagen. Puede ser relativa o absoluta.
  • `alt` (alternative text): Texto descriptivo de la imagen. ¡Es crucial para la accesibilidad (lectores de pantalla) y el SEO!
  • `width` y `height`: Atributos opcionales para definir las dimensiones de la imagen en píxeles. Es mejor controlarlas con CSS para responsividad.

Puedes usar la etiqueta `<figure>` y `<figcaption>` para imágenes con leyendas, dándoles un significado semántico más profundo.


<figure>
    <img src="https://via.placeholder.com/300x200" alt="Ejemplo de una imagen placeholder">
    <figcaption>Figura 1: Una imagen de ejemplo.</figcaption>
</figure>

Creación de Tablas HTML

Las tablas se usan para mostrar datos tabulares (filas y columnas), no para el diseño de la página. Utilizan varias etiquetas:

  • <table>: El contenedor principal de la tabla.
  • <thead>: Agrupa el contenido del encabezado de la tabla.
  • <tbody>: Agrupa el contenido del cuerpo de la tabla.
  • <tfoot>: Agrupa el contenido del pie de la tabla.
  • <tr>: Define una fila de la tabla (Table Row).
  • <th>: Define una celda de encabezado (Table Header Cell).
  • <td>: Define una celda de datos (Table Data Cell).
  • <caption>: Proporciona un título para la tabla.
  • colspan: Atributo para que una celda ocupe varias columnas.
  • rowspan: Atributo para que una celda ocupe varias filas.

<table>
    <caption>Ejemplo de Tabla de Productos</caption>
    <thead>
        <tr>
            <th>Producto</th>
            <th>Precio</th>
            <th>Stock</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Laptop</td>
            <td>$1200</td>
            <td>15</td>
        </tr>
        <tr>
            <td>Teclado</td>
            <td>$75</td>
            <td>50</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="3">Datos actualizados al mes.</td>
        </tr>
    </tfoot>
</table>

Diseño de Formularios HTML

Los formularios permiten a los usuarios interactuar con tu sitio web, enviar datos e información. El elemento principal es `<form>`.

  • <form action="procesar.php" method="POST">: El contenedor del formulario.
    • `action` (acción): La URL a la que se envían los datos del formulario.
    • `method` (método): El método HTTP para enviar los datos (`GET` o `POST`). `POST` es preferible para datos sensibles o grandes.
  • <label for="id-input">: Una etiqueta descriptiva para un control de formulario. El atributo `for` debe coincidir con el `id` del control.
  • <input type="text" id="nombre" name="nombre" placeholder="Tu nombre">: El elemento de entrada de datos más versátil.
    • `type` (tipo): Define el tipo de control (text, password, email, number, checkbox, radio, submit, etc.).
    • `id` (identificador): Un identificador único para el elemento (usado con `<label>`).
    • `name` (nombre): El nombre que se usará para identificar los datos enviados al servidor.
    • `placeholder` (marcador de posición): Texto de ejemplo dentro del campo.
    • `required` (requerido): Hace que el campo sea obligatorio.
  • <textarea name="comentarios" rows="4" cols="50"></textarea>: Un área de texto multi-línea.
  • <select name="opcion"> y <option>: Para listas desplegables.
  • <button type="submit">Enviar</button>: Un botón para enviar el formulario.
  • <fieldset> y <legend>: Agrupan lógicamente controles de formulario y proporcionan un título para el grupo.

<form action="/submit-form" method="POST">
    <fieldset>
        <legend>Información Personal</legend>
        <p>
            <label for="nombre-usuario">Nombre:</label>
            <input type="text" id="nombre-usuario" name="nombreUsuario" required>
        </p>
        <p>
            <label for="email-usuario">Email:</label>
            <input type="email" id="email-usuario" name="emailUsuario" placeholder="ejemplo@dominio.com">
        </p>
    </fieldset>

    <p>
        <label for="mensaje-usuario">Mensaje:</label><br>
        <textarea id="mensaje-usuario" name="mensajeUsuario" rows="5" cols="40"></textarea>
    </p>

    <p>
        <label for="pais">País:</label>
        <select id="pais" name="pais">
            <option value="es">España</option>
            <option value="mx">México</option>
            <option value="co">Colombia</option>
        </select>
    </p>

    <button type="submit">Enviar Formulario</button>
</form>
