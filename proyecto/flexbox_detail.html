Flexbox (Flexible Box Layout): Diseño Unidimensional

Flexbox es un módulo de diseño de CSS que proporciona una forma eficiente de distribuir y alinear elementos dentro de un contenedor, incluso cuando su tamaño es desconocido o dinámico. Es particularmente útil para la creación de componentes y para el diseño en una sola dimensión (fila o columna).

Conceptos Clave de Flexbox

  • Contenedor Flex (Flex Container): Es el elemento padre al que le aplicas `display: flex;` o `display: inline-flex;`.
  • Ítems Flex (Flex Items): Son los hijos directos del contenedor flex.
  • Eje Principal (Main Axis): La dirección principal en la que los ítems flex se distribuyen. Por defecto, es horizontal (`row`).
  • Eje Cruzado (Cross Axis): La dirección perpendicular al eje principal. Por defecto, es vertical (`column`).

Propiedades del Contenedor Flex

  • display: flex; o display: inline-flex;: Habilita el contexto flex.
  • flex-direction: row | row-reverse | column | column-reverse;: Define la dirección del eje principal.
  • justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;: Alinea los ítems a lo largo del eje principal.
  • align-items: flex-start | flex-end | center | baseline | stretch;: Alinea los ítems a lo largo del eje cruzado.
  • flex-wrap: nowrap | wrap | wrap-reverse;: Controla si los ítems deben ajustarse a una nueva línea.
  • align-content: flex-start | flex-end | center | space-between | space-around | stretch;: Alinea las líneas de ítems flex dentro del contenedor flex cuando hay varias líneas (solo si `flex-wrap` es `wrap`).

Propiedades de los Ítems Flex

  • flex-grow: <number>;: Define la capacidad de un ítem para crecer si hay espacio extra.
  • flex-shrink: <number>;: Define la capacidad de un ítem para encogerse si no hay espacio suficiente.
  • flex-basis: <length> | auto;: Define el tamaño por defecto de un ítem antes de que se distribuya el espacio restante.
  • flex: <flex-grow> <flex-shrink> <flex-basis>;: Atajo para las tres propiedades anteriores.
  • align-self: auto | flex-start | flex-end | center | baseline | stretch;: Sobreescribe la propiedad `align-items` para un ítem individual.
  • order: <integer>;: Controla el orden visual de los ítems flex.

.flex-example {
    display: flex;
    justify-content: space-evenly; /* Espacio uniforme entre y alrededor */
    align-items: center; /* Alinea verticalmente al centro */
    height: 150px; /* Para visualizar align-items */
    background-color: #f0f8ff;
    border: 1px solid #cceeff;
}
.flex-ex-item {
    background-color: #6a5acd; /* Púrpura oscuro */
    color: white;
    padding: 20px;
    margin: 5px;
    border-radius: 5px;
}
