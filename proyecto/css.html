CSS: Estilizando tu Contenido

CSS (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación de un documento escrito en HTML. Permite controlar el color, la fuente, el espaciado, el diseño, y mucho más, de tus páginas web.

Conceptos Clave de CSS

  • Selectores: Se utilizan para "seleccionar" los elementos HTML a los que quieres aplicar estilo (por ejemplo, `p` para párrafos, `#id-elemento` para un ID específico, `.clase-elemento` para una clase).
  • Propiedades y Valores: Las reglas CSS se componen de una propiedad (lo que quieres estilizar, como `color` o `font-size`) y un valor (cómo quieres estilizarlo, como `blue` o `16px`).
  • Modelo de Caja: Cada elemento HTML se considera una caja. El modelo de caja describe cómo se distribuyen el contenido, el padding (relleno), el border (borde) y el margin (margen) alrededor de cada elemento.
  • Casacada y Especificidad: Determinan qué estilos se aplican cuando múltiples reglas CSS pueden afectar el mismo elemento.

Formas de Incluir CSS

  1. Estilos en línea: Directamente en el atributo `style` de una etiqueta HTML (no recomendado para grandes proyectos). 
    <p style="color: blue;">Este es un párrafo azul.</p>
  2. Estilos internos: Dentro de la etiqueta `<style>` en el `<head>` del documento HTML (útil para estilos específicos de una sola página). 
    
<head>
    <style>
        h1 {
            color: green;
        }
    </style>
</head>
  3. Estilos externos (Recomendado): En un archivo CSS separado (como `styles.css`) enlazado desde el HTML. Esto permite reutilizar estilos en múltiples páginas y mantener el código más limpio. 
    <link rel="stylesheet" href="styles.css">
