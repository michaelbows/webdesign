CSS Grid Layout: Diseño Bidimensional

CSS Grid Layout es un sistema de diseño bidimensional para la web. Te permite diseñar interfaces de usuario complejas y robustas controlando tanto las columnas como las filas de una vez. Es ideal para el diseño de la estructura general de una página o secciones principales.

Conceptos Clave de CSS Grid

  • Contenedor Grid (Grid Container): El elemento padre al que le aplicas `display: grid;` o `display: inline-grid;`.
  • Ítems Grid (Grid Items): Los hijos directos del contenedor grid.
  • Líneas Grid (Grid Lines): Las líneas horizontales y verticales que forman la estructura de la cuadrícula.
  • Pistas Grid (Grid Tracks): Los espacios entre las líneas grid, es decir, las columnas y filas.
  • Celdas Grid (Grid Cells): La intersección de una fila y una columna, similar a una celda de tabla.
  • Áreas Grid (Grid Areas): Un área rectangular definida por la unión de varias celdas grid.

Propiedades del Contenedor Grid

  • display: grid; o display: inline-grid;: Habilita el contexto grid.
  • grid-template-columns: <track-list>;: Define las columnas de la cuadrícula.
  • grid-template-rows: <track-list>;: Define las filas de la cuadrícula.
  • grid-template-areas: <string>+;: Define las áreas de la cuadrícula usando nombres.
  • gap: <gap-value>; (o `grid-gap` obsoleto): Espacio entre las celdas grid (filas y columnas). También `row-gap` y `column-gap`.
  • justify-items: start | end | center | stretch;: Alinea el contenido dentro de las celdas a lo largo del eje de la fila.
  • align-items: start | end | center | stretch;: Alinea el contenido dentro de las celdas a lo largo del eje de la columna.
  • justify-content: start | end | center | space-between | space-around | space-evenly;: Alinea la cuadrícula entera a lo largo del eje de la fila dentro del contenedor grid.
  • align-content: start | end | center | space-between | space-around | space-evenly;: Alinea la cuadrícula entera a lo largo del eje de la columna dentro del contenedor grid.

Propiedades de los Ítems Grid

  • grid-column-start: <line>; / grid-column-end: <line>;: Dónde comienza/termina el ítem en las columnas.
  • grid-row-start: <line>; / grid-row-end: <line>;: Dónde comienza/termina el ítem en las filas.
  • grid-column: <start> / <end>;: Atajo para las propiedades de columna.
  • grid-row: <start> / <end>;: Atajo para las propiedades de fila.
  • grid-area: <name> | <row-start> / <column-start> / <row-end> / <column-end>;: Define el área de un ítem, o lo asigna a un área con nombre.
  • justify-self: start | end | center | stretch;: Alinea un ítem individual a lo largo del eje de la fila.
  • align-self: start | end | center | stretch;: Alinea un ítem individual a lo largo del eje de la columna.

Ejemplo Práctico de CSS Grid

Layout de 2 columnas y 2 filas con áreas nombradas:

Header
Sidebar
Content
Footer

.grid-example {
    display: grid;
    grid-template-columns: 1fr 3fr; /* 1 columna pequeña, 1 grande */
    grid-template-rows: auto 1fr auto; /* Header, Contenido, Footer */
    grid-template-areas:
        "header header"
        "sidebar content"
        "footer footer";
    gap: 10px;
    height: 300px;
    background-color: #f8f8ff;
    border: 1px solid #e0e0f0;
}
.header { grid-area: header; background-color: #4682b4; color: white; } /* Azul acero */
.sidebar { grid-area: sidebar; background-color: #6a5acd; color: white; } /* Púrpura oscuro */
.content { grid-area: content; background-color: #87ceeb; color: #333; } /* Azul cielo */
.footer { grid-area: footer; background-color: #4682b4; color: white; }
.grid-ex-item {
    padding: 15px;
    text-align: center;
    border-radius: 5px;
    display: flex;
    justify-content: center;
    align-items: center;
}
